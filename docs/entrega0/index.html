DESCRIPCIÓN

Prototipo de alarma para cuando se le acabe el agua al perro

DESCRIPCIÓN

Con plasticina conductiva se realizó un prototipo de alarma para el agua del perro, en donde cuando se acabe el agua la luz del led se apagará de igual forma. Esto es gracias que el agua sirve como conductor y, por ende, cuando el agua baja la corriente no puede pasar al otro lado.
